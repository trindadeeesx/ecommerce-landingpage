# ecommerce-landingpage

Demo de landing page para produto — React + Tailwind + Framer Motion.  
Inclui: hero, cards de produto, formulário de checkout simulado.

## Rodar local
1. `npm install`  
2. `npm run dev`

## Estrutura
- `src/components` — Button, Card, CardContent  
- `src/pages/LandingPage.jsx` — página principal

## Observações
Demo pronta para adaptação e integração com gateway de pagamento (Pix, Stripe) e backend (Node/Express).
